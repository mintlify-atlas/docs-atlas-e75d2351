---
title: "Quickstart — Run AlonsoChatBot in Under Two Minutes"
description: "Get AlonsoChatBot running in five steps: grab a free Groq API key, download the project, open index.html, enter your key, and send your first message."
---

AlonsoChatBot has no build system or dependencies to install. The entire setup is three files — `index.html`, `app.js`, and `style.css` — and runs in any modern browser. Follow the steps below to go from zero to your first AI conversation in under two minutes.

<Steps>
  <Step title="Get a free Groq API key">
    Go to [console.groq.com/keys](https://console.groq.com/keys) and sign in or create a free account. Click **Create API key**, give it a name, and copy the generated key. It starts with `gsk_`.

    Keep this key somewhere safe — the console only shows it once.
  </Step>

  <Step title="Download the project">
    Clone the repository or download it as a ZIP file from GitHub.

    <CodeGroup>
      ```bash Clone with Git
      git clone https://github.com/aloncm/chatbot.git
      cd chatbot
      ```

      ```bash Download ZIP (no Git required)
      # Visit https://github.com/aloncm/chatbot
      # Click Code → Download ZIP, then unzip the folder
      ```
    </CodeGroup>
  </Step>

  <Step title="Open the app in your browser">
    You can open `index.html` directly by double-clicking it, or serve the folder with a local static server if your browser restricts `file://` requests.

    <CodeGroup>
      ```bash Python (built-in, no install needed)
      python -m http.server 8000
      ```

      ```bash Node.js (via npx, no global install needed)
      npx serve .
      ```
    </CodeGroup>

    If you use a local server, open `http://localhost:8000` (Python) or the URL printed by `npx serve` in your browser.
  </Step>

  <Step title="Enter your API key in the setup overlay">
    The first time you open the app, a welcome overlay appears asking for your Groq API key. Paste the `gsk_...` key you copied in step 1 into the input field and click **Activate My ChatBot**.

    The app validates that the key starts with `gsk_` and saves it to `localStorage` under `GROQ_API_KEY`. The overlay closes automatically on success.

    <Tip>
      Because the key is stored in `localStorage`, you only need to enter it once per browser. Refreshing or reopening the tab will not prompt you again — the app reads the saved key automatically. To use a different key, open the browser's developer tools, clear `localStorage`, and reload.
    </Tip>
  </Step>

  <Step title="Send your first message">
    Type any message in the input field at the bottom of the chat window and press **Enter** or click the send button. A typing indicator appears while the Groq API processes your request, then the bot's reply slides in.

    The full conversation is kept in memory for the current session, so the model has context of everything said so far. To start fresh, open **Settings** (the gear icon in the header) and click **Clear Chat History**.
  </Step>
</Steps>

<Check>
  If you see a reply from AlonsoChatBot, everything is working correctly. You now have a fully functional, serverless AI chatbot running entirely in your browser.
</Check>
