AlonsoChatBot is a lightweight, browser-based AI chatbot that connects directly to the [Groq API](https://console.groq.com) using the `llama-3.1-8b-instant` model. It runs entirely in your browser — no server required — and features a sleek glassmorphism UI, persistent session memory, and a fully customizable bot personality.

## How it works

AlonsoChatBot sends your messages to the Groq API along with a running conversation history, so the bot remembers the context of your entire session. Your API key is stored locally in your browser and never leaves your machine.

<Steps>
  <Step title="Get a free Groq API key">
    Visit [console.groq.com/keys](https://console.groq.com/keys) and create a free API key. It starts with `gsk_`.
  </Step>
  <Step title="Open the chatbot">
    Open `index.html` in your browser — or serve the project with any static file server.
  </Step>
  <Step title="Enter your API key">
    Paste your Groq API key into the setup overlay. It is saved to `localStorage` so you only need to do this once.
  </Step>
  <Step title="Start chatting">
    Type a message and press **Send**. The bot responds in real time using LLaMA 3.1.
  </Step>
</Steps>
