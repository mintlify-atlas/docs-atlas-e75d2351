AlonsoChatBot is a pure browser application that lets you have real-time conversations with a LLaMA 3.1-powered AI assistant — no server, no build step, and no package manager required. You open `index.html`, paste your Groq API key once, and the bot is ready to chat. Your key is saved in `localStorage` so you never have to enter it again, and a full session conversation history is maintained in memory so the bot remembers everything you've said during the current tab session.

## How it works

When you load the page, `app.js` checks `localStorage` for a stored key under `GROQ_API_KEY`. If no key is found, a setup overlay appears asking you to paste your key. Once saved, every message you type is appended to the `messages[]` array and sent to the Groq API using the `llama-3.1-8b-instant` model at `temperature: 0.7`. The response is appended to both the array (for future context) and the chat UI (for display).

<Note>
  A Groq API key is completely free. You can create one at [console.groq.com/keys](https://console.groq.com/keys) with no credit card required. Free-tier rate limits are generous enough for personal and development use.
</Note>

## Prerequisites

Before you start, make sure you have:

- A modern browser (Chrome, Firefox, Safari, or Edge)
- A free Groq API key (starts with `gsk_`)
